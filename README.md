## 🚀 How to Run Locally

1. **Clone the repository**
   ```bash
   git clone https://github.com/<your-username>/portfolio.git
   cd portfolio
   ```

2. **Open in browser**
   ```bash
   # Simply open the file directly — no server needed
   open index.html
   ```
   Or drag and drop `index.html` into any browser.

---

## 🌍 Deploy on GitHub Pages

1. Push the project to a GitHub repository
2. Go to **Settings → Pages**
3. Under **Source**, select `main` branch → `/ (root)`
4. Click **Save**
5. Your site will be live at:
   ```
   https://<your-username>.github.io/<repo-name>/
   ```

## ✏️ Customization

To personalize the portfolio, open `index.html` and update:

| What to Change | Where to Find It |
|---|---|
| Email address | `#contact` section → `href="mailto:..."` |
| GitHub link | `#contact` section → GitHub anchor tag |
| LinkedIn link | `#contact` section → LinkedIn anchor tag |
| Skill percentages | `#skills` section → `style="width:XX%"` |
| Bio text | `#about` section → paragraph tags |
| Footer year | `<footer>` tag at the bottom |
